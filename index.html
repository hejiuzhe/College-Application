🚀 配置向导

跟着步骤走，5 分钟完成 高考志愿填报 Agent 的搭建

1. 环境准备

安装项目运行所需的两个基础工具

安装 Bun（JavaScript 运行时）

Bun 负责编译 TypeScript、安装依赖、启动服务。

powershell -c "irm bun.sh/install.ps1 | iex"
📋 如何执行：
1右键点击屏幕左下角 开始菜单（田字图标）
2选择 "终端" 或 "Windows PowerShell"
3点击上方代码块的 📋 复制 按钮，然后 Ctrl + V 粘贴到终端
4按 Enter 键执行命令
5安装完成后关闭终端重新打开，输入 bun --version 验证。显示版本号则无误可关闭；显示"不是内部命令"则需检查是否下载成功，或手动将 Bun 添加到系统 PATH
💡安装后输入 bun --version 看到版本号即成功。

下载 Qdrant（向量数据库）

从 GitHub 下载 Windows 平台的 qdrant 可执行文件。

找到 qdrant-x86_64-pc-windows-msvc.zip → 下载 → 解压到任意目录（比如 D:\qdrant），里面有 qdrant.exe。

📝记住你解压的目录路径，步骤 4 启动 Qdrant 时需要用到。

→ GitHub Releases 下载页面

2. 获取项目

克隆仓库并安装依赖

克隆代码

git clone https://github.com/Zeke-777/gaokao-agent.git cd gaokao-agent
📋 如何执行：
1打开终端，输入 cd D:\ 按 Enter（换成你想放的目录）
2复制第一行命令粘贴，按 Enter 克隆仓库
3再复制第二行，按 Enter 进入项目目录
4⚠️ 后续所有命令都要在这个目录下执行！
💡没装 Git？到 GitHub 页面点 "Download ZIP" 下载解压，然后终端 cd 到解压目录即可。

安装依赖

bun install
📋 怎么执行：粘贴命令到终端，按 Enter 执行，等待依赖安装完成（约 10-30 秒）。

3. 配置环境变量

在下方表单填写 API Key，然后点击按钮直接写入项目根目录的 .env 文件

🧠 LLM 大模型 必填

👉 到 platform.deepseek.com 创建 Key。也支持其他 OpenAI 兼容接口。
💰 充值 2-3 块钱即可，够用很久。
可选 deepseek-v4-pro、deepseek-v4-flash、deepseek-chat

🔢 Embedding 向量化 必填

👉 到 dashscope.console.aliyun.com 创建 Key（新用户有免费额度）。💰 即使付费，2-3 块钱也绰绰有余。

🔍 搜索引擎 必填

👉 到 app.tavily.com 创建 Key（月免 1000 次，免费额度足够日常使用）。

⚙️ 其他 可选

# 填写上方表单后，这里自动生成 .env 内容，点击下方按钮写入项目根目录

4. 启动 Qdrant

启动向量数据库，默认监听 localhost:6333

D:\qdrant\qdrant.exe --config-path qdrant\config\qdrant.yaml

⚠️ 把 D:\qdrant\ 换成你在步骤 1 解压的实际目录

📋 如何执行：
1右键开始菜单，打开 "终端" 或 "PowerShell"
2先进入项目目录：输入 cd D:\gaokao-agent（改成你的实际路径），按 Enter
3复制上方命令，把路径换成步骤 1 解压的目录，按 Enter 执行
4看到日志输出说明启动成功。⚠️ 不要关闭这个终端窗口！
🌐访问 http://127.0.0.1:6333，看到 JSON 包含 "title":"qdrant" 就是成功了。
⚠️ 保持这个终端开着，别关。
⚠️常见问题：端口被占用 → 先关掉占用程序。Windows 用户确认 qdrant.exe 和 qdrant\config\ 都在项目根目录。

5. 初始化知识库

打开一个新终端，进入项目根目录，依次执行下面三条命令：

📋 操作说明：
1新开一个终端窗口（不要用步骤 4 正在运行 Qdrant 的那个窗口）
2先进入项目目录：输入 cd D:\gaokao-agent（改成你的实际路径），按 Enter
3依次复制下方三条命令，每条执行完等它结束再执行下一条
4每次粘贴后按 Enter 键执行

① 创建集合

bun run qdrant/scripts/create-collections.cjs

创建 schools、majors、policies_rules、province_data、style_cases 五个集合

② 同步 Wiki 数据

bun run qdrant/scripts/sync-wiki-to-staging.ts

将 wiki/ 下的 Markdown 复制到 staging 目录并切分段落

③ 导入向量库

bun run qdrant/scripts/ingest-staging.cjs

对文本生成 Embedding 写入 Qdrant（首次约 1-3 分钟，取决于 API 速度）

⏱️执行完根目录会出现 .qdrant-initialized 标记文件，表示初始化成功。过程约 1-3 分钟，运行中不要关闭终端，执行完再关。

6. 启动服务 🎉

打开一个新终端，进入项目根目录，依次执行：

bun run build bun run server
📋 操作说明：
1打开一个新终端，先进入项目目录：输入 cd D:\gaokao-agent（改成你的实际路径），按 Enter
2粘贴第一行 bun run build，按 Enter 执行构建
3构建完成后，粘贴第二行 bun run server，按 Enter 启动服务
4看到 🚀 高考志愿Agent API 字样说明启动成功。⚠️ 不要关闭这个终端窗口！
